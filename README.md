# Mkulimaa-AI 🌱

An AI-powered voice and text agent built to help small-scale farmers in Kenya get instant, localized advice on crop health and market trends.

## Built At
Safaricom Decode Event — built as part of a hands-on activity during the event

## What It Does
- Provides real-time crop health advice
- Acts as a market negotiator to help farmers get fair prices
- A farmer tells the AI what they have and the AI suggests the best day to sell and a fair price based on current market trends
- Supports both voice and text interaction

## Tech Stack
- TypeScript
- Google AI Studio (Gemini)

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies: `npm install`
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app: `npm run dev`
